3.請寫一段 Ruby 程式，計算檔案 `homework-03-31-content.txt` 中有幾個字的長度是小於或等於 5 個字母。

Ans:
```ruby
count = 0
f = File.open("homework-2016-03-31-content.txt")
  f.each do |f1|
    f1.delete(".,").split(" ").each do |line|
      count += 1 if line.length <= 5
    end
  end
f.close
puts count
```
