## answer
```ruby
content = ""
File.open('homework-2016-03-31-content.txt'){|f|
	f.each_line{|line|
		content += "#{f.lineno} #{line}"
	}
}

File.open('result.txt','w'){|f|
	f.puts content
}
```

3.請寫一段 Ruby 程式，計算檔案 `homework-03-31-content.txt` 中有幾個字的長度是小於或等於 5 個字母。  
```ruby
File.open('homework-2016-03-31-content.txt'){|f|
```
