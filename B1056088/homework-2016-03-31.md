3.請寫一段 Ruby 程式，計算檔案 `homework-03-31-content.txt` 中有幾個字的長度是小於或等於 5 個字母。

程式是：

```
w = 0
File.open("homework-2016-03-31-content.txt") do |f|
 f.read.split(" ").each do |word|
   w_len = word.delete("." ",")
   w += 1 if w_len.length <= 5
 end
end
print w, "個字長度小於等於5\n"
```
