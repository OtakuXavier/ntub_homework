* 請印出在 1 到 2000 中，可以被 4 整除，但不能 100 整除，但又要可以被 400 整除的數字。(潤年計算公式)
```
list = (1..2000).to_a.select{ |num| num % 4 == 0}.reject{ |num| num % 100 == 0} + (1..2000).to_a.select{ |num| num % 400 == 0}
p list.sort
```
